推理测试代码
参数定义在"config/conf_xx.yaml"

提供两种测试方法
---------------------------------------------
****1、test.py #比对两条语音相似度得分
---------------------------------------------
输入：
需要提供测试文件trials，以及训练好模型路径（model_path） model_path/10.params
预训练模型和测试trials在data中，打分文件和eer文件在exp中
目前提供模型为resnet，训练数据为voxceleb2_dev,在voxceleb1_O中测试EER(%)=2.10

trials:<label wav1_path wav2_path>
label值为0或1
0=不同说话人；1相同

输出：
得分文件scores.txt
eer.txt

---------------------------------------------
****2、test_model_test.py #测试语音与每个注册说话人模型相似度得分
---------------------------------------------
根据给定注册文件提取说话人注册模型，之后逐个比对测试语音与说话人模型得分
输入：
需要提供注册文件，测试文件
注册文件：<说话人ID 注册语音1 注册语音2 ... 注册语音n>
测试文件：<测试语音>

---------------------------------------------

路径需要注意
trials中语音路径为相对路径，语音文件所在路径需要提供，<testset_root>
打分文件输出路径<save_path>

执行方式：
python test.py
